# ETF Review OS — Dutch terminology preferences

Use this file to keep the Dutch companion report natural, premium, and consistent.

## Core rule
The Dutch report should read like high-quality Dutch written for serious subscribers.
It must preserve the exact substance of the canonical English report, but should avoid awkward literal translations.

## Preferred translations
- sleeve -> bouwsteen
- lane -> thema
- carry-forward -> doorgeschoven waardering
- funded switch -> gefinancierde wissel
- resilience -> weerbaarheid or veerkracht
- actionable now -> nu inzetbaar
- scale in slowly -> rustig opbouwen
- watchlist -> watchlist
- too early -> te vroeg
- pullback -> terugval
- breadth -> breedte
- leadership -> leiderschap
- holdings -> posities
- core beta -> kernblootstelling brede markt

## Style reminders
- Keep tickers exact.
- Keep numbers exact.
- Prefer natural Dutch over Anglo-Dutch hybrids.
- Keep the title `Weekly ETF Pro Review` in English unless explicitly changed later.
- Keep table and section structure aligned with the English canonical report.
